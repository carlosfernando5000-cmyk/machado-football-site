# Machado Football Site

Site simples da loja Machado Football.

## Para editar
No arquivo `src/App.jsx`, troque:
- `55SEUNUMEROAQUI` pelo seu número de WhatsApp com DDD.
- textos, preços e categorias.
- link do Instagram, se precisar.

## Para publicar
1. Suba essa pasta no GitHub.
2. Entre no Vercel.
3. Clique em Add New Project.
4. Importe o repositório.
5. Clique em Deploy.
